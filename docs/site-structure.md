# Структура сайта барбершопа

## Полный список страниц проекта

| № | Название страницы | URL | Краткое описание |
|---|-------------------|-----|------------------|
| 1 | Главная | / | Приветствие, акции, лучшие мастера, отзывы |
| 2 | Услуги и цены | /services | Список всех услуг с ценами и временем |
| 3 | Мастера | /barbers | Карточки мастеров, их описание и расписание |
| 4 | Портфолио работ | /portfolio | Фотографии стрижек до/после |
| 5 | Отзывы | /reviews | Отзывы клиентов с оценками |
| 6 | Контакты | /contacts | Адрес, карта, телефон, график работы |
| 7 | Запись онлайн | /booking | Форма записи к мастеру |
| 8 | Регистрация | /register | Форма создания аккаунта |
| 9 | Вход | /login | Форма авторизации |
| 10 | Личный кабинет | /profile | Дашборд пользователя |
| 11 | История записей | /profile/history | Список всех посещений |
| 12 | Избранные мастера | /profile/favorites | Сохранённые мастера |
| 13 | Настройки профиля | /profile/settings | Редактирование данных |
| 14 | Админ-панель | /admin | Главная панель управления |
| 15 | Управление услугами | /admin/services | CRUD услуг |
| 16 | Управление мастерами | /admin/barbers | CRUD мастеров |
| 17 | Управление записями | /admin/bookings | Календарь всех записей |
| 18 | Управление отзывами | /admin/reviews | Модерация отзывов |
| 19 | Статистика и отчёты | /admin/statistics | Доходы, загрузка мастеров |
| 20 | Страница 404 | /404 | Страница не найдена |
