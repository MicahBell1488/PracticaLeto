# Пользовательские сценарии

## Сценарий 1: Гость хочет записаться на стрижку

**Участник:** Гость (незарегистрированный клиент)

**Предусловие:** Гость открыл сайт барбершопа

**Ход событий:**

1. Гость видит главную страницу с услугами
2. Нажимает кнопку "Записаться"
3. Заполняет форму:
   - Имя: "Алексей"
   - Телефон: "+7 999 123-45-67"
   - Выбирает услугу: "Мужская стрижка"
   - Выбирает мастера: "Дмитрий"
   - Выбирает дату: "Завтра"
   - Выбирает время: "14:00"
4. Нажимает кнопку "Записаться"

**Результат:**
- Гость видит сообщение: "Запись принята! Ждём вас завтра в 14:00"
- Гость получает SMS-напоминание за час до записи
- Запись сохраняется в систему

**Альтернативный сценарий (ошибка):**
- Если время уже занято, гость видит: "Это время занято. Выберите другое"
- Если телефон не указан, форма не отправляется и показывает ошибку

## Сценарий 2: Пользователь хочет отменить запись

**Участник:** Зарегистрированный пользователь

**Предусловия:**
- Пользователь авторизован в системе
- У пользователя есть активная запись на завтра в 14:00

**Ход событий:**

1. Пользователь заходит в личный кабинет
2. Видит свои активные записи
3. Находит запись на завтра в 14:00
4. Нажимает кнопку "Отменить"
5. Система спрашивает: "Вы уверены?" → пользователь нажимает "Да"
6. Система удаляет запись

**Результат:**
- Запись исчезает из списка активных
- Пользователь видит сообщение: "Запись отменена"
- Пользователю приходит SMS или email: "Запись отменена"

**Альтернативный сценарий (поздно отменять):**
- Если до записи осталось меньше 2 часов, кнопка "Отменить" неактивна
- При попытке отмены система пишет: "Отмена невозможна. До записи менее 2 часов"

## Сценарий 3: Администратор добавляет нового мастера

**Участник:** Администратор

**Предусловия:**
- Администратор авторизован
- Администратор находится в админ-панели

**Ход событий:**

1. Администратор нажимает раздел "Мастера"
2. Нажимает кнопку "Добавить мастера"
3. Заполняет поля:
   - Имя: "Антон"
   - Фото: загружает файл
   - Описание: "Мастер спорта по барберским дисциплинам"
   - Расписание: ПН-ПТ с 10:00 до 19:00, СБ с 11:00 до 17:00
4. Нажимает кнопку "Сохранить"

**Результат:**
- Новый мастер появляется в общем списке
- Мастер становится доступен для записи клиентов
- Администратор видит сообщение: "Мастер Антон добавлен"

**Альтернативный сценарий (не все поля заполнены):**
- Если не заполнено имя, система пишет: "Заполните имя мастера"
- Сохранение не происходит
